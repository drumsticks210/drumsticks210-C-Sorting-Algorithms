Contents of this Zip file

1. ReadMe.txt - This document
2. Prog4Main.cpp - The file you can use as the base for your program 4.
3. MicroSecondTimer.h - Header file to use in timing the sorts.  You must use
	this as the standard timing in milliseconds is not fast enough to time
	some of the sorts.
4. MicroSecondTimer.cpp - The implementation file for the microsecond timer.
5. SortData.txt - A text file full of data to use for testing.  This contains 15,000 records
